extra-1.6.18: Extra functions I use.

Safe HaskellSafe
LanguageHaskell2010

Data.IORef.Extra

Description

This module extends Data.IORef with operations forcing the value written to the IORef. Some of these functions are available in later versions of GHC, but not all.
